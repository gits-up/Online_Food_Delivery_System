# Online_Food_Delivery_System
A fullstack online food delivery system. 

#### Technology (mern)

- React
- node js
- express js
- role base access (ROLE_CUSTOMER, ROLE_RESTAURANT_OWNER)
- nodemailer
- jeson web token
- mongoDB(Data base)
- Tailwind css
- Mui (css component library)
- Redux (State managment library)
- Axios
- strip payment gatway
- Spring boot
- spring security
- spring start mail
- Mysql(Data base)
